API Documentation

Welcome to the API documentation page.

The API provides endpoints to manage users and cards in a system. It includes routes for CRUD operations on users and cards, along with middleware for logging, error handling, and CORS support.


Base URL http://localhost:8080/api/v1

Users:

Endpoints to manage users.

  • GET /users: Get all users.
  • POST /users: Create a new user.
  • DELETE /users/:id: Delete user by ID.
  • PUT /users/:id: Update user by ID.
  • POST /users/login: Login user.
  • PATCH /users/:id: Change businees status.
  • GET /users/:id: Get user by ID.

For detailed request and response examples, refer to the Postman documentation: Users Postman Collection

Cards:

Endpoints to manage cards.

  • POST /cards: Create a new card.
  • GET /cards/:id: Get card by ID.
  • GET /cards/my-cards: Get my-cards.
  • GET /cards: Get all cards.
  • PUT /cards/:id: Update card by ID.
  • PATCH /cards/:id: Like card.
  • DELETE /cards/:id: Delete card by ID.

For detailed request and response examples, refer to the Postman documentation: Cards Postman Collection
